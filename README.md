CypherX ni WhatsApp Bot ya kisasa iliyojengwa kwa Node.js inayowezesha matumizi ya AI, automation ya majibu, na amri mbalimbali zinazorahisisha mawasiliano. Imetengenezwa kwa lengo la kusaidia mawasiliano ya haraka, burudani, na usimamizi wa makundi ya WhatsApp.

Vipengele Muhimu:
- Auto-reply & AI Chat
- Media downloader (YouTube, Instagram)
- Group moderation tools
- Custom commands
- Owner & sudo access
- Anti-link, welcome & goodbye msgs

Jinsi ya Kuitumia:
1. Clone repo hii
2. Tumia hosting (Heroku, Railway, Termux, nk)
3. Set `.env` zako (SESSION, OWNER...)
